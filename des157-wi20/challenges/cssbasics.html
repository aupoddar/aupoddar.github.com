Inheritance in CSS

Inheritance in CSS is when a value is specified for a property on an element. In other words, unless you've applied a property directly to an element, a property will be styled with whatever element it is under (if under one)

Specificity in CSS

Specificity in CSS means that the browser decides which CSS rule is more specific and applies it. Therefore, if there are more than 2 conflicting rules, the browser will decide what will be applied.

ID and Class Attributes

In HTML, an id attribute is a special identifier which is used to perform a specific task for an element. To access the id attribute in CSS, you can use the # symbol and the name of the id. A class attribute is used when wanting to specifiy one or more class names. This is how you can perform specific tasks to elements that have the class identifier.
